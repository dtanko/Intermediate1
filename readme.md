**Assignment Operators**
    Equal(=)
    Addition(+=)
    Subtraction(-=)

    Division(/=)
    Remainder(%=)
    Exponential(**=)
    Left shift (<<=)

**Comparison Operators**
    Equal(==)
    Not equal(!=)
    Strict equal(===)
    Strict not equal(!==)
    Greater than(>)
    Less than(<)
    Greater than or equal(>=)
    Greater than or equal(<=)

**Examples**
        1. 3 == var1
            "3" == var1

        2. var1 !== "3"

**Examples**
    1. const a1 = true && true; // t && t returns true
